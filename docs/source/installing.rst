========================
Installing
========================


Installing with pip
========================

We recommend using pip to install btrdb-python on all platforms:

.. code-block:: bash

    $ pip install btrdb

To get a specific version of btrdb-python:

.. code-block:: bash

    $ pip install btrdb==4.0.0


To upgrade using pip:

.. code-block:: bash

    $ pip install --upgrade btrdb
